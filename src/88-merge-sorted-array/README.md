# 35.Merge Sorted Array

## Description

Given two sorted integer arrays nums1 and nums2, merge nums2 into nums1 as one sorted array.

## Note

You may assume that nums1 has enough space to hold additional elements from nums2.

## Example1

```javascript
Input: [1, 3, 5][(2, 3, 6)]
Output: [1, 2, 3, 3, 5, 6]
```
