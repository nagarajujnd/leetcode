# 95.Valid Perfect Square

## Description

Given a positive integer num, write a function which returns True if num is a perfect square else False.

## Example1

```javascript
Input: 16
Returns: True
```

## Example2

```javascript
Input: 14
Returns: False
```

## Note

* Do not use any built-in library function such as sqrt.
