# 77.Reverse Vowels of a String

## Description

Write a function that takes a string as input and reverse only the vowels of a string.

## Example1

```javascript
Input: 'hello'
Output: 'holle'
```

## Example2

```javascript
Input: 'leetcode'
Output: 'leotcede'
```
