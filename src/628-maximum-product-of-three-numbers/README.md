# 106.Maximum Product of Three Numbers

## Description

Given an integer array, find three numbers whose product is maximum and output the maximum product.

## Example1

```
Input: [1,2,3]
Output: 6
```

## Example2

```
Input: [1,2,3,4]
Output: 24
```
