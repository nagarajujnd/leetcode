# 79.Valid Anagram

## Description

Given two strings s and t, write a function to determine if t is an anagram of s.

## Example1

```javascript
Input: 'anagram', 'nagaram'
Output: true
```

## Example2

```javascript
Input: 'rat', 'car'
Output: false
```
