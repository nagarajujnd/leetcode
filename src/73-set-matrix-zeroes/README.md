# 2.Set Matrix Zeroes

## Description

Given a m x n matrix, if an element is 0, set its entire row and column to 0. Do it in-place.

## Example1

```js
Input: [[1, 1, 1], [1, 0, 1], [1, 1, 1]]
Output: [[1, 0, 1], [0, 0, 0], [1, 0, 1]]
```

## Example2

```js
Input: [[0, 1, 2, 0], [3, 4, 5, 2], [1, 3, 1, 5]]
Output: [[0, 0, 0, 0], [0, 4, 5, 0], [0, 3, 1, 0]]
```
